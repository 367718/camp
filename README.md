# camp

## Binaries

* **minami**: files and lists manager.
* **nadeshiko**: torrent files downloader.
* **aoi**: mpv player remote controller.

## Libraries

* **rin**: configuration files manager.
* **ena**: files manager.
* **chiaki**: lists manager.
* **akari**: HTTP/2 client.
* **chikuwa**: collection of small utilities.
